# ☕ Cafeteria - Code & Café

Este projeto consiste em um website moderno e responsivo para uma cafeteria fictícia chamada **Code & Café**. Foi desenvolvido com foco em **HTML5** e **CSS3** para praticar a criação de layouts atraentes, estruturação semântica e design adaptável a diferentes dispositivos.

O site apresenta diversas seções informativas, um menu de produtos, depoimentos de clientes e informações de localização, proporcionando uma experiência completa ao usuário.

## 📖 Seções disponíveis

- 🏠 Home (Introdução e Chamada para Ação)
- ℹ️ Sobre (História e Diferenciais da Cafeteria)
- ☕ Menu (Lista de Produtos com Preços)
- ⭐ Avaliações (Depoimentos de Clientes)
- 📍 Endereços (Localização com Mapa Interativo)

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3 (Flexbox e Grid)
- Google Fonts (Fonte Roboto)
- Icons8 (Ícones de Busca, Carrinho e Redes Sociais)

## 📚 Conceitos praticados

- Estruturação semântica de páginas web com HTML5.
- Estilização avançada com CSS3, utilizando variáveis para gerenciamento de cores.
- Criação de layouts flexíveis e responsivos utilizando Flexbox e Grid CSS.
- Integração de fontes externas via Google Fonts.
- Inserção de ícones externos para aprimorar a interface do usuário.
- Incorporação de mapas interativos via iframe.
- Design de interface focado em usabilidade e estética moderna.

## 🎯 Objetivo

O principal objetivo deste projeto foi aplicar e consolidar conhecimentos em desenvolvimento front-end na construção de um site comercial completo. A prática envolveu desde a organização da estrutura do conteúdo até a estilização detalhada de cada componente, garantindo um visual coeso e funcional em diversas resoluções de tela.
